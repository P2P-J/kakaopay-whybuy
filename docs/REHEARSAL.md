## 3. 케이스 리허설

### 3.1 case-001 — 삼성전자 (유효 판정·하락·나빠진 공시 0건)

- **매수 2025-03-25 @ 59,800원 → as_of 2025-04-09 @ 53,000원 (−11.4%)**
- **매수 논거(실적 개선):** 사업보고서 2024.12 (rcp `20250311001085`) — 2024년 전 분기 영업이익 대폭 개선(20244Q 32.7조, 전년 6.6조 대비 +398%).
- **판정(RULE-EARN-01):** as_of 시점 최신 잠정실적 = 20251Q 연결영업(잠정) (rcp `20250408800003`, 2025-04-08), 영업이익 6.7조 vs 전년 6.6조 **YoY +1% → 개선=유효**. CFS 기준, 동일 보고서 종류끼리 비교.
- **나빠진 공시 0건:** 매수창(2025-02-23~04-09) 공시는 전부 지분(D)·거버넌스(사외이사/대표이사변경)·IR·정기보고서. **자기주식처분·감자·CB·소송·감액 잠정실적 없음** → 반증/약화 공시 0.
- **장면 성립:** "가격은 −11% 빠졌지만 산 이유(실적 개선)는 공시상 여전히 유효" — 처분효과 반박 케이스.

### 3.2 case-002 — 카카오 (급변–공시 매칭·약화/반증·거울)

- **매수 2025-02-24 @ 39,550원 → as_of 2025-05-14 @ 37,950원 (−4.0%)** *(승인 후 마진 조정, §0 참조)*
- **매수 논거(실적 개선):** 20244Q 연결영업(잠정) (rcp `20250213800004`, 2025-02-13) — 2024년 개선세(1Q +92%/2Q +18%/3Q +5%).
- **거울 약화(RULE-EARN-01):** as_of 최신 잠정실적 = **20251Q 연결영업(잠정) (rcp `20250508800012`, 2025-05-08), 영업이익 1,054억 vs 전년 1,203억 → YoY −12% = 감익 → 반증/약화.** 매수 논거와 **동일한 "영업(잠정)실적" 공시 유형**이 as_of에 반대 신호로 재등장 = 거울.
- **보조 약화:** 전환사채발행결정 (rcp `20250522000332`, 2025-05-22) — 희석 요인(주주가치 반증).

> **⚠ 논점(아엔 판단 요청):** case-002의 "급변–공시 매칭"이 이 창에서는 약하다. 카카오의 실제 급변일(±5% 이상, 총 43건)은 **대부분 상승 랠리**(예: 2025-05-27 +5.0%, 2025-06-09 +16.0%, 2025-08-07 +12.0%)이고, **약화 신호인 20251Q 감익 잠정실적(5/8)은 −3.5%로 급변 문턱 미달**이다. 즉 "논거 약화"와 "가격 급변"이 이 종목·기간에서 같은 날 겹치지 않는다.
>
> **선택지:**
> - **(A) 현 제안 유지** — case-002를 "실적 거울 약화" 중심으로 하고, 급변일 타임라인은 별도 표기(약화와 별개의 시장 이벤트로 정직하게 제시). 데이터 왜곡 없음.
> - **(B) 급변+악재 동시성 재탐색** — 다른 종목/기간에서 하락 급변일과 반증 공시가 겹치는 창을 재수집. (대형주 특성상 희소, 추가 수집 필요)
>
> 리허설 권고: **(A)**. "급변과 논거는 항상 같은 날 겹치지 않는다"는 것 자체가 시스템의 정직한 메시지(급변≠논거변화)와 부합.

- **추천: KT&G.** RULE-DIV-01(주당배당금 유지·증가=유효) 판정이 5,000→5,200→5,400→6,000 단조 증가로 **가장 깨끗한 유효**. 비금융이라 영업이익 계정 의미 왜곡 없음(금융지주 안전핀 불필요). 배당 주기 단순(연 1회).

- **KB 참고:** 증가 추세로 유효 판정은 가능하나, 금융지주라 `verdict_engine`의 영업이익 안전핀 로직을 실제로 타야 함 → 케이스 단순성 목적엔 KT&G가 우위. (KB는 향후 "금융지주 안전핀" 회귀 테스트 픽스처로 보존 권장.)
